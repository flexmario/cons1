1. Как работает Web

Описание схемы:

  1. Клиент вводит запрос в браузере и нажимает Enter
  2. Браузер отправляет запрос пользователя в библиотеку DNS
  3. Браузер получает из DNS значение IP
  4. Браузер отправляет IP на сервер, на котором хранится страница
  5. Сервер отправляет браузеру файлы страницы (HTML, CSS и др.)
  6. Браузер рендерит (обрабатывает) полученные файлы и выводит их на экран пользователю
